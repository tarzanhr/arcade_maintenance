# Minesweeper

## Description
Minesweeper est un casse-tête classique revisité pour l'arcade de l'IUTLCO.  Votre mission : démêler un champ miné, où chaque case peut cacher une menace explosive ! Un véritable défi de logique et d'analyse où chaque clic est crucial. 

## Objectif du jeu
Le but du jeu est simple : révéler toutes les cases sans toucher aux mines cachées. Pour réussir, vous devrez déduire la position des mines en observant les nombres affichés sur certaines cases. Ces chiffres indiquent le nombre de mines présentes dans les cases adjacentes. Soyez stratégique, chaque mouvement compte !

## Comment jouer
Contôlez votre explorateur avec les joysticks pour naviguer dans le champ miné. Utilisez le bouton "A" pour creuser une case et découvrir son contenu, ou le bouton "B" pour y placer un drapeau pour marquer une case suspectée d'abriter une mine. Le bouton "C" permet de recommencer la partie si vous êtes trop imprudent !

## Points forts
Minesweeper offre un gameplay addictif qui stimule votre cerveau. Ses graphismes colorés et son ambiance rétro-futuriste rendent l'expérience ludique et agréable. La possibilité de choisir différents niveaux de difficulté vous permettra de tester vos compétences à votre rythme. 

## Aspect technique
Le jeu a été développé en Java, offrant une performance fluide et des graphismes attractifs. L'intégration d'une musique spatiale immersive et d'effets sonores retro-gaming complète l'expérience.

## Pour qui ?
Minesweeper est idéal pour tous les amateurs de casse-têtes logiques et stratégiques. Que vous soyez un joueur débutant ou expérimenté, le jeu offre des défis stimulants à chaque niveau.

## Note finale
Préparez-vous à affronter un champ miné palpitant ! Minesweeper vous fera vivre une expérience captivante et addictive.  Testez vos réflexes et votre logique, devenez un expert du déminage !
